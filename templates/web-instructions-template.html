Intégration de l'outil d'analytique et de publicité à votre site Web
Pour être en mesure de vous présenter des statistiques complètes et pertinentes, vous devez effectuer les étapes suivantes pour chacune de vos pages HTML.
Ajouter le module d'analytique
Le module d'analytique est un fichier JavaScript qui nous permet de faire le suivi de vos visiteurs et de leurs afficher des publicités ciblées. Ajouter la ligne suivante à la toute fin de votre <body>. 
<script src="https://s3.amazonaws.com/www.squidsquads.site/js/analytique.js"></script>
Ajouter vos identifiants personnalisés
Grâce à ces identifiants, nous sommes en mesure d'identifier votre site parmi tous les autres. Il suffit de rajouter les 4 balises <meta> suivantes dans le <head> de vos pages HTML. 
<meta name="squidsquads-user-id" content="{{userId}}">
<meta name="squidsquads-banner-hor" content="{{horId}}">
<meta name="squidsquads-banner-ver" content="{{verId}}">
<meta name="squidsquads-banner-mob" content="{{mobId}}">
Afficher des publicités
Si vous souhaitez afficher des publicités, il suffit d'ajouter des balises <img> de la même façon qu'elles sont présentées ci-dessous. Vous pouvez afficher une, deux ou trois publicités de différentes tailles.
Vous ne pouvez pas utiliser plus d'une fois un même type de publicité par page (i.e. au maximum une publicité horizontale, verticale et mobile).
Le standard des tailles des publicités est disponible ici. 
<img id="squidsquads-img-hor" src="" alt=""/>
<img id="squidsquads-img-ver" src="" alt=""/>
<img id="squidsquads-img-mob" src="" alt=""/>
